THANK YOU FOR CHECKING OUT LEASE LEMUR!

Welcome to the premiere commerical real estate lease tracking system on the block!

Code Repository
It can be be found on Github. The application was built off of a MEAN.js scaffolding.
Here's what it can do right now!
  • Login in as user 'laura.lemur' and password 'wiredtri'. Or you can created a new email-based account. (The social account APIs are not currently configured).
  • View a list of all existing Leases
  • View an existing Lease
  • Create a new Lease. One leasing period will be created by default, initially matching the duration of the Lease. Multiple leasing periods can be added.
  • Edit Leases that you have created
  • Deleting Leases that you have created
  • Download a CSV of all Leases (basic fields only)
